# ⚔️ RPG Character Generator

### 🎮 [Live Demo](http://alogcoden.github.io/-RPG-Character-Generator-Educational-Project---Beginners-/)

A beginner-friendly Python project that simulates the creation of an RPG character through user interaction. Users provide a character name, city, and class, and the program generates a formatted character sheet while demonstrating fundamental Python concepts such as input handling, variables, string manipulation, and built-in functions.

---

## ✨ Key Features

* Character creation through user input
* Dynamic character sheet generation
* Character name length calculation
* Clean and structured console output
* Beginner-friendly Python implementation
